nombre de la aplicación: nosVemos
nombre del dominio: https://nosvemos-mx.com/
lenguaje de programación: Vite + React + JavaScript


Idea proyecto:

Crear una pagina web que funcione en laptops, tablets, moviles que pueda crear invitaciones digitales por medio de template generales.
Enfoque: las invitaciones estan enfocadas en bodas, quinceaños, fiestas y evento corporativos.

la pagina tiene que ser amigable e intuitiva donde el usuario pueda de alguna manera sencilla seleccionar un template y pueda 
previsualizar un ejemplo generico ya sea para boda, quinceaños, evento, etc. dependiendo a la categoria seleccionada, que sea un tipo
hybrido donde el usuario pueda seleccionar un template y podamos tener un formulario donde llenariamos los componentes necesarios para la invitación.



Fase 1: me gustaria empezar con un layout tipo Hero + Galería de plantillas + Layout tipo Landing ultra simple - 
    la idea son 3 paquetes diferentes, cada uno con mas caracteristicas que el siguiente con precioes que van entre los $999, $1500, $1850 MXN
    el usuario podra ver los templates por categoria y la pantalla principal tipo giro podra mostrar como las diferentes opciones y paquetes
